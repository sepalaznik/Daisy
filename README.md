# Макет лендинговой страницы "Daisy"

Автор: Сергей Палазник, 2021

Проект на основе свободного макета. В процессе работы изучено и использовано несколько способов
создания шестиугольников (с использованием маски, с использованием прямоугольника и двух псевдотреугольников,
с клонированием и поворотом прямоугольника); слайдер с использованием js, а также замедленная прокрутка 
содержимого страницы; вместо модального окна использована анимация карты с указанием локации; 
и применено несколько лёгких анимационных эффектов по наведению мышки.

Проект проверен на соответствие макету при помощи PerfectPixel, а также адаптирован при помощи медиазапросов.
Не для ie :)
